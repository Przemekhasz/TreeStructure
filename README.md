## To require node_modules

```js
npm update
```

## Make migration tables to database

```php
php artisan migrate:fresh
```

## Run server

```php
php artisan serve
```

## Clear routing cache

```php
php artisan route:cache
```

## View routing lists

```php
php artisan route:list
```
